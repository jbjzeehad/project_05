Scope of Var,Let and Const

Scope means where variables are available for use. The scope is global when a var variable is declared outside a function. It means any variable that is declared with var outside a function block is available for use in the whole window. It is available and can be accessed only within the function.
let is block scoped. A block is a chunk of code bounded by {}. When a variable declared in a block with let is only available for use within that block.
Like let, const declarations can only be accessed within the block they were declared. But const can not be updated or re-declared.

Use case of null and undefined

In JS, null is a special value that represents the absence of an object. It is not same as an empty string or zero value. On the other hand, undefined is a special value that is assigned to a variable by default. The main difference between null and undefined is that undefined used to check if a variable has been assigned a value or not. Where null is a value that can be assigned to a variable.

REST API

API stands for Application Programming Interface. An API acts like a link that allows two applications to talk to eaxh other. It is the part of the server that receives requests and sends response. REST stands for REpresentational State Transfer. REST APIs make the communication process with the server easier for us by giving us various HTTP request methods. The most commonly used methods are GET, POST, PATCH/PUT, DELETE. These methods provides by REST allow us perform CRUD operations easily.
